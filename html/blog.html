Git and Github

  • Git is a Distributed Version Control System (DVCS)
  • Basic Git commands
    1. Initialize: create new git repository (.git directory)
    2. Add: updates index and prepare content in staging area
    3. Commit: takes snapshot of changes and update repository
    4. Pull: fetches changes from remote repository i.e. github
    5. Push: commits local repository to remote
  • Advanced Git commands
    1. Branching: allows for parallel development through separate lines of development
    2. Merging: integrate change from one branch into another
    3. Rebasing: move starting point of branch to different commit
